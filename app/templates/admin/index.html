Welcome, {{g.user.username}}, to the Admin Page


This page for admins only. It is designed to be able to create and manage quizzes with their respective multiple choice, short answer and long answer questions. This page also allows you to manipulate the information of a User and of Posts where you see fit.
There are also functionalities which allow you to assess responses to long answer questions, provide feedback on them, then update a Users marks accordingly. Above you will see a navigation bar leading to tables of stored data through which you are able to efficiently search through to find specific users, quizzes, questions, marks or posts.


Users Table

The users table is able to be refined to find specific users with a given username or email.


Post

The post table contains the posts and how they connected to the user and the quiz it was posted underneath.


Questions

The question table allows you to view questions and which quiz they are allocated to. You also are able to add questions to quizzes yourself. When making a question you are able to assign yourself as the authour of the quiz. Questions must be brief and could contain both worded or numeral solutions.


Multiple Choice

The multichoice table allows you to view questions and which quiz they are allocated to. You also are able to add questions to quizzes yourself. When making a question you are able to assign yourself authour of the quiz. Questions can be assigned options, with one as the solution, and may contain both worded or numeral responses.


Long Answer

The longanswer table allows you to view questions and which quiz they are allocated to. You also are able to add questions to quizzes yourself. When making a question you are able to assign yourself authour of the quiz. There are no solutions as these are manually graded. Your are able to filter this table by both quiz, user or question for your ease.


Quiz Marks

The quizMarks table allows you to see marks of users for each quiz. You are able to edits marks for users and provide feedback if it is for a long worded response.
